## 🐛 前端可能的问题

### 问题 1：Axios 超时设置过短

**原因**：
- PDF 525 KB → Base64 700 KB
- 通过公网传输需要 15+ 秒
- Axios 默认超时可能 < 15 秒
- 导致请求被取消，显示错误

**影响**：
- 大文件上传失败
- 小文件可能正常

### 问题 2：错误信息不准确

**原因**：
- Axios 捕获的错误可能是 `ECONNABORTED` 或 timeout
- 但前端显示为"文件过大"
- 给用户错误的提示

### 问题 3：浏览器缓存旧版本

**原因**：
- 修改后的前端代码可能还在浏览器缓存
- 导致使用旧的超时设置

## 🔧 修复方案

### 修复 1：增加 Axios 超时设置

**文件**: `frontend/src/App.tsx`

**当前**：
```typescript
const response = await axios.post(`${API_URL}/upload-pdf`, {
    pdf_base64: base64
});
```

**修改为**：
```typescript
const response = await axios.post(`${API_URL}/upload-pdf`, {
    pdf_base64: base64
}, {
    timeout: 120000  // 120 秒超时
});
```

### 修复 2：增加错误处理

**修改为**：
```typescript
try {
    const response = await axios.post(`${API_URL}/upload-pdf`, {
        pdf_base64: base64
    }, {
        timeout: 120000
    });

    setPages(response.data.pages);
    await detectQuestions(response.data.pages);
} catch (error: any) {
    console.error('Upload failed:', error);
    
    // 详细的错误信息
    if (error.code === 'ECONNABORTED') {
        alert('上传超时，请检查网络连接或重试');
    } else if (error.response?.status === 413) {
        alert('文件过大，请上传更小的 PDF');
    } else {
        alert(`上传失败: ${error.message || '未知错误'}`);
    }
}
```

### 修复 3：添加上传进度显示

**修改为**：
```typescript
const [uploadProgress, setUploadProgress] = useState(0);

const handlePdfUpload = async (event: React.ChangeEvent<HTMLInputElement>) => {
    const file = event.target.files?.[0];
    if (!file) return;

    // 显示文件大小
    const fileSizeKB = (file.size / 1024).toFixed(2);
    console.log(`File size: ${fileSizeKB} KB`);

    const reader = new FileReader();
    reader.onload = async (e) => {
        const base64 = e.target?.result as string;

        try {
            const response = await axios.post(`${API_URL}/upload-pdf`, {
                pdf_base64: base64
            }, {
                timeout: 120000,
                onUploadProgress: (progressEvent) => {
                    const percentCompleted = Math.round(
                        (progressEvent.loaded * 100) / progressEvent.total
                    );
                    setUploadProgress(percentCompleted);
                    console.log(`Upload progress: ${percentCompleted}%`);
                }
            });

            setPages(response.data.pages);
            await detectQuestions(response.data.pages);
            setUploadProgress(0);  // 重置进度
        } catch (error: any) {
            console.error('Upload failed:', error);
            setUploadProgress(0);
            
            if (error.code === 'ECONNABORTED') {
                alert('上传超时，请检查网络连接或重试');
            } else if (error.response?.status === 413) {
                alert('文件过大，请上传更小的 PDF');
            } else {
                alert(`上传失败: ${error.message || '未知错误'}`);
            }
        }
    };
    reader.readAsDataURL(file);
};
```
